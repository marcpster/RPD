## 🚀 Getting Started

### 📦 Dependencies

- next 14.0+
- node v20.10+
- npm v10.2+
- tailwind v3.3+

### 👉 Install Dependencies

```bash
npm install
```

### 👉 Development Command

```bash
npm run dev
```

### 👉 Build Command

```bash
npm run build
```
